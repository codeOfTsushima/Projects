# Message Bomb

A Flask-based web application that lets users send temporary, self-destructing messages. Built with a sleek, neon-green aesthetic, this project ensures your messages vanish after a set period, leaving no trace behind.

## Features

* **Self-Destructing Messages:** Messages automatically expire and become inaccessible after a designated timeframe.
* **Custom Timers:** Granular control over how long a message remains visible to the recipient.
* **Sender ID Handling:** Robust tracking and management of sender identities for secure message routing.
* **Cyberpunk Aesthetic:** A responsive, custom-styled UI featuring a distinct neon-green color palette.

## Technologies Used

* **Backend:** Python, Flask
* **Frontend:** HTML5, CSS3 

## Getting Started

### Prerequisites
Make sure you have Python installed on your system. 

### Installation

1. Clone this repository:
   ```bash
   git clone [https://github.com/codeOfTsushima/message-bomb.git](https://github.com/codeOfTsushima/message-bomb.git)
   cd message-bomb

2. Install the required dependencies:
```Bash
    pip install Flask
```
3. Run the application: 
```Bash
    python app.py 
```  
4. Open your browser and navigate to http://127.0.0.1:5000/ to use the app.
